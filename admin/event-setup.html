What this page does today

This is the organizer bridge between the web and mobile experience. It does not yet write to Supabase. It standardizes the event ID, app join payload, and on-site QR workflow so the public event pages and app flow can stay aligned.

Creates a real app payload

The generated value follows your current iOS parsing logic: beacon://event/<event-id>.

Supports the physical event flow

Organizers can print or display this QR code in the room so attendees can join without needing a laptop.

Prepares for future integration

This same structure can later save events to Supabase and feed public event pages, recaps, and analytics.
